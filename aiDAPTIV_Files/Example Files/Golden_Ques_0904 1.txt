===== Alpha無限公司 員工獎懲管理辦法.pdf ===== 
公司級獎勵的提名時間是哪一季？
→ 第四季，為期一個月。

Level 3 的懲處是否需要委員會審議？
→ 是，需由懲處審議委員會投票決定。

書面警告的影響時效是多久？
→ 12個月。

誰有權解釋這份獎懲辦法的條文？
→ 人力資源暨組織發展部。

重大違紀（Level 4）是否一定會被解僱？
→ 通常會終止勞動合約，並可能追究法律責任
 

===== Alpha無限公司 員工晉升管理辦法.pdf =====
標準晉升週期一年有幾次？
→ 兩次（上半年 H1、下半年 H2）。

員工多久可以具備晉升提名資格？
→ 任職滿 12 個月，且績效達「符合期望」以上。

是否可以自我提名晉升？
→ 可以，限 L5 以上職級，且需主管提供評語。


===== Alpha無限公司 員工考績管理辦法.pdf =====
績效評估一年進行幾次？
→ 兩次（7月與次年1月）。

績效評級共有幾個等級？
→ 五級（從「待改進」到「重新定義卓越」）。

PIP 的執行期限多久？
→ 30 至 60 天。

 
===== Alpha無限公司 出勤與休假管理規章.pdf =====
家庭照顧假？
→ 有薪，每年七天，不併入事假計算。

生日假可以怎麼休？
→ 生日當月任選一天休假。

病假是否需要醫療證明？
→ 單日不需，連續或累計三日以上需提供。

志工服務假一年有幾天？
→ 五個工作天，全薪。


===== Alpha無限公司 人才招聘、任用與發展綜合管理規章.pdf =====
內部職缺需先公告幾天？
→ 至少五個工作日。

員工多久後可申請內部轉調？
→ 任職滿十二個月。

離職預告期最長是多少天？
→ 任職三年以上者需提前 30 天。
