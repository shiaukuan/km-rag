===== The_Shawshank_Redemption_1994.zh-TW.srt =====
安迪入獄前的職業是什麼？
→ 波特蘭銀行的副總裁。

安迪請瑞德幫他弄進監獄的第一樣東西是什麼？用途為何？
→ 一把鶴嘴槌（rock hammer），用來收集和打磨石頭，這是他的嗜好。

安迪因為播放莫扎特音樂被關禁閉多久？他認為值得嗎？
→ 被關禁閉兩周。他認為那是他最好的時光，因為有莫扎特陪伴。


===== The_Godfather_1972.zh-TW.srt =====
為什麼包納瑟拉在柯里昂女兒的婚禮當天來求教父幫忙？
→ 因為沒有西西里人能在女兒結婚那天拒絕任何的請求。

湯姆·哈金與柯里昂家族是什麼關係？他在家族中擔任什麼角色？
→ 他小時候被發現流浪街頭，被教父收養。他是家族的軍師（顧問），地位非常重要。

麥克在餐廳刺殺索拉索的計畫中，槍藏在哪裡？
→ 藏在餐廳廁所的舊式馬桶水箱裡，用膠帶綁著。


===== The_Godfather_Part_II_1974.zh-TW.srt =====
年幼的維多·柯里昂為何必須逃離西西里島？
→ 他的父親因侮辱當地黑手黨老大被殺，兄長保羅發誓復仇後也失蹤。黑手黨老大西西歐擔心維多長大後會回來報仇，下令殺他，母親犧牲自己讓他逃走。

在教父第二集中，范倫奇是什麼人？他對鄰里做了什麼？
→ 范倫奇是當地的黑手黨，他向鄰里的意大利人強行收取保護費，後來還要求收雙倍的錢。

葛瑞參議員對麥克·柯里昂提出的賭場執照要求是什麼？
→ 要求25萬元加上每月四家旅館合併總收入的5%，遠超過執照本身不到兩萬元的價格，因為他意圖壓迫柯里昂家族。


===== The_Dark_Knight_2008.zh-TW.srt =====
電影開場的銀行搶劫中，小丑的計畫有什麼特別之處？
→ 小丑安排每個劫匪在完成任務後殺掉另一個同夥，逐步減少分錢的人數，最終由他獨吞所有贓款。

哈維·乃特用來做決定的硬幣有什麼來歷？
→ 那是他父親的幸運幣，也促成了他和瑞秋的第一次約會。

布魯斯·韋恩為什麼要求改進他的蝙蝠裝？
→ 因為盔甲太重了，他需要變得更快。福克斯先生負責幫他改進裝備。


===== 12_Angry_Men_1957.zh-TW.srt =====
陪審團第一次投票的結果是什麼？
→ 十一票有罪，一票無罪。

八號陪審員如何反駁「凶刀獨一無二」的說法？
→ 他前一晚到被告住的社區附近的當鋪，花了六塊錢買到一把與凶刀完全相同的摺疊刀，證明那把刀並非獨一無二。

被告聲稱案發當晚他去了哪裡？為什麼這個說法受到質疑？
→ 被告聲稱十一點半出門去看電影，凌晨三點十分回家。但沒有人看到他離家，電影院也沒人見過他，而且他完全不記得電影的名字。
